# Type

Type is a simple and fast Markdown editor.

Typeはシンプルで軽快なMarkdownエディターです。

  • Support Extension
  • Easy preview
  • Auto saving
  • Easy type Markdown
  • Support 3D touch
  • Searchable with Spotlight
  • Support post to Qiita
  • Extension対応
  • 素早いプレビュー(入力画面・一覧画面)
  • 自動保存
  • ツールバーからMarkdownを簡単入力
  • 3Dタッチのサポート
  • Spotlightからの検索に対応
  • Qiitaへの投稿対応
